　コードに関しては、セクションや入力フォームの項目等が増減した場合でも変更しやすいように
　 HTML の構造をまとめたり、CSS の共通部分は流用しやすいように工夫しました、が、
　そのために想定通りに表示されないことが多々あり、結果的に余計に時間がかかってしまいました。
